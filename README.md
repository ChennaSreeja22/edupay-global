# 🌍 EduPay Global

EduPay Global is a fintech MVP designed to simplify international education payments for students and their families. The application provides a simple and transparent way for parents to send tuition fees and living expenses abroad while tracking transfers in real time.

This project was developed as the Week 4 Capstone Project for **Product Management 101** in Learners' Space.

---

## 🚀 Features

- User onboarding
- KYC verification
- Dashboard with account overview
- Send Money workflow
- Fee and transfer summary
- Transfer status tracking
- Transaction history
- Notifications
- User profile

## 👨‍👩‍👧 Demo Scenario

The application demonstrates the journey of **Ramesh Kumar**, who uses EduPay Global to send tuition fees to his daughter **Priya Ramesh**, a student studying at the **University of Manchester, United Kingdom**.

The user journey includes:

- Creating an account
- Completing KYC verification
- Accessing the dashboard
- Sending an international payment
- Tracking the transfer
- Viewing transaction history and notifications

## ▶️ Running the Project

Clone the repository

```bash
git clone <repository-url>
```

Go to the project folder

```bash
cd edupay-global
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
